Difference between HTML and HTML5
Parameters HTML HTML5
Multimedia Does not support audio and video without the use of flash player support. Supports audio and video controls with the use of <audio>and <video> tags.
JavaScript Does not allow JavaScript. Allows JavaScript.
drag-and-drop Does not allow drag-and-drop effects. Allows drag-and-drop effects.
Shapes Not possible to draw shapes like circles, rectangles, triangles, etc. Allows to draw shapes like circles, rectangles, triangles, etc.
Doctype declaration too long and complicated. quite simple and easy.
Elements Elements like nav, header were not present. New elements for web structurer like nav, header, footer are present.
Character encoding long and complicated. Simple and easy
