1. HTML

    2. Primary Use Cases Key features/Benefits
    Standard markup language for creating Web pages.
    • Light weighted and fast to load.
    • Simple to edit as being a plain text.
    • Easy to learn

  2. CSS

    3. Primary Use Cases Key features/Benefits
    Used to style a Web page.
    • Applied consistently across variety of sites.
    • Simplifies the maintenance.
    • Less complex therefore the effort are significantly reduced.
    • Power for re-positioning.
    • Reduces the file transfer size.

  3. JS(JavaScript)

    4. Primary Use Cases Key features/Benefits
    To build web-based applications
    • Save lots of bandwidth and make the execution process fast.
    • Ability to support all modern browsers and produce an equivalent result.
    • Employed everywhere on the web.
    • Gives the power to make rich interfaces.
